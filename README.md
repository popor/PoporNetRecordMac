# PoporNetRecordMac

功能：
1：可以配合iOS:PoporNetRecord一起工作，将网络请求、异常、重要数据发送给PoporNetRecordMac（接口方法可以查看代码中的新增方法）。
2：可以在请求详情页面重新发起网络请求，用于快速调试代码。
3：可以生成测试请求连接，在后台接口不完善的情况下正常工作，也可以使用EOlinker，但是没有本功能灵活。

使用限制：需要在同一局域网内。
